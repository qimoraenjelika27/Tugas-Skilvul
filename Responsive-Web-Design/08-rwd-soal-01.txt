1. Apa itu responsive web design?
Jawaban :
Responsive web design atau desain web responsif adalah sebuah teknik atau metode bagi web designer 
untuk membuat suatu layout website yang dapat menyesuaikan diri sesuai dengan ukuran layar pengguna.
