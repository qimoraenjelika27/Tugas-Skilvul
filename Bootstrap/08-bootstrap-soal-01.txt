1. Apa itu CSS Framework?
Jawaban :
CSS framework adalah kerangka kerja yang fungsinya untuk memudahkan Anda ketika mendesain website atau aplikasi, tepatnya di bagian layout dan tema. CSS framework berisi kumpulan kode CSS yang sudah siap pakai. Anda tak perlu repot untuk membuat struktur dasar desain dari nol.
